Introduction

This is a developer documentation for the GeoMoS Monitor API Version 1. Initially call http://localhost:8000/v1 in a browser to find out if the api is available, which is the case if the GeoMoS controller service is running. The answer should include the version number of service, like this: GeoMoS RestApi V1. To access GeoMoS projects, you must first activate the option 'Web API' in the project properties of GeoMoS Monitor. After that type this in the browser: http://localhost:8000/v1/projects You should see the list of projects where you activated the option 'Web API'. Continue then with the list of points of one specific project, like: http://localhost:8000/v1/projects/3005/points (put in the id of the project, in this example I put in 3005).
